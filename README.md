# ([dannotharmarajah.dev](https://dannotharmarajah.dev/))

My personal portfolio — built with a Windows 95 aesthetic because why not?

## What is this?

A single-page portfolio showing my freelance work, hobby projects, school projects, and links to find me elsewhere. Designed to feel like a desktop from 1995, with draggable-style windows, a scrolling marquee, and hover image previews on the project listings.

## Built with

- **React** + **TypeScript** — component structure and type safety
- **Vite** — dev server and build tooling
- **Tailwind CSS** — styling and responsive layout
- **GSAP** — powers the marquee animation at the top

## Design

The UI is inspired by Windows 95 — grey beveled borders, blue gradient title bars, inset text areas, and a teal desktop background. On mobile the windows stack vertically. On desktop they're positioned like open windows on a screen.
